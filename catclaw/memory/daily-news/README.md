# OpenClaw 每日资讯收集方案

## 已实现的功能

### 1. 自动搜索
- 每天定时搜索 OpenClaw 相关新闻和技术分享
- 多关键词覆盖："OpenClaw"、技术分享、教程等
- 搜索来源：互联网最新资讯

### 2. 智能筛选
- **去重**：自动检测重复 URL，避免重复内容
- **质量评分**：
  - 🟢 高质量 (80-100分)：权威来源、技术深度、内容完整
  - 🟡 中质量 (60-79分)：一般来源、内容尚可
  - 🔴 低质量 (0-59分)：已自动过滤
- **评分维度**：
  - 来源权威性（GitHub、技术博客加分）
  - 内容长度（太短的内容减分）
  - 技术关键词（教程、架构、部署等加分）
  - 重复内容（大幅减分）

### 3. 结构化输出
- Markdown 表格格式
- 包含：质量标签、标题（可点击跳转）、来源域名、摘要
- 每日统计：各质量等级数量汇总

### 4. 定时推送
- 每天上午 9:00 自动执行
- 结果通过消息推送给你
- 同时保存到 `memory/daily-news/YYYY-MM-DD.md`

## 定时任务信息

| 项目 | 值 |
|------|------|
| 任务名称 | OpenClaw 每日资讯收集 |

## 管理命令

```bash
# 查看定时任务状态
openclaw cron list

# 手动触发一次
openclaw cron run 8b9c9f42-a638-44f8-84d2-4dcc03a00e0

# 修改时间（例如改成晚上8点）
openclaw cron update 8b9c9f42-a638-44f8-84d2-4dcc03a00e0 --schedule "0 20 * * *"

# 删除任务
openclaw cron remove 8b9c9f42-a638-44f8-84d2-4dcc03a00e0
```
