# React Best Practices CN - 安装说明

## 技能包信息

- **名称**: react-best-practices-cn
- **版本**: 1.0.0
- **语言**: 中文
- **类型**: 客户端 React 性能优化指南
- **基于**: Vercel React Best Practices（精简版）

## 快速安装

### 方式 1：使用 zip 文件

1. 解压 `react-best-practices-cn.zip`
2. 将解压后的 `react-best-practices-cn` 文件夹复制到你的技能目录

### 方式 2：直接使用

如果你已经在 `skills` 目录中，技能包已经可以使用了。

## 文件说明

```
react-best-practices-cn/
├── AGENTS.md              # 完整的编译文档，包含所有规则
├── SKILL.md               # 技能定义和快速参考
├── README.md              # 项目说明和贡献指南
├── CHANGES.md             # 详细的变更说明
├── 安装说明.md            # 本文件
├── metadata.json          # 元数据配置
└── rules/                 # 规则文件目录
    ├── _sections.md       # 章节定义
    └── *.md               # 33 个规则文件
```

## 使用方法

### 在 AI 助手中使用

1. **引用整个技能包**：
   ```
   @react-best-practices-cn
   ```

2. **引用特定规则**：
   ```
   @react-best-practices-cn/rules/bundle-barrel-imports.md
   ```

3. **引用完整文档**：
   ```
   @react-best-practices-cn/AGENTS.md
   ```

### 在代码审查中使用

AI 助手会自动应用这些规则来：
- 审查你的 React 代码
- 提供性能优化建议
- 重构现有代码
- 生成符合最佳实践的新代码

## 规则分类

### 1. 消除瀑布流（2条）
关键性能优化，消除顺序 await 导致的延迟。

### 2. 包体积优化（3条）
减少初始包体积，改善加载时间。

### 3. 客户端数据获取（4条）
优化数据获取，减少冗余请求。

### 4. 重渲染优化（7条）
减少不必要的重渲染，提高响应性。

### 5. 渲染性能（6条）
优化渲染过程，减少浏览器工作量。

### 6. JavaScript 性能（9条）
热路径微优化，累积性能提升。

### 7. 高级模式（2条）
特定场景的高级优化技巧。

## 适用场景

✅ **适用于**：
- Create React App 项目
- Vite + React 项目
- 单页应用（SPA）
- 客户端渲染应用
- React Router 应用

❌ **不适用于**：
- Next.js 应用（请使用原始版本）
- 服务端渲染应用
- React Server Components
- API 路由开发

## 与原版的区别

1. **移除了服务端相关内容**：
   - 整个"服务端性能"章节
   - Next.js 特定优化
   - RSC（React Server Components）
   - API 路由优化

2. **精简了复杂规则**：
   - 移除了依赖较少的库（如 better-all）
   - 移除了实验性 API（如 Activity 组件）
   - 保留了核心和常用的优化规则

3. **完全中文化**：
   - 所有说明文本翻译为中文
   - 保持代码示例的英文注释
   - 术语符合中文技术社区习惯
