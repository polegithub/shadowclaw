# 章节

此文件定义所有章节、排序、影响级别和描述。
章节 ID（括号中）是用于分组规则的文件名前缀。

---

## 1. 消除瀑布流 (async)

**影响：** 关键  
**描述：** 瀑布流是性能杀手第一名。每个顺序 await 都会增加完整的网络延迟。消除它们能带来最大收益。

## 2. 包体积优化 (bundle)

**影响：** 关键  
**描述：** 减少初始包体积可以改善交互时间和最大内容绘制。

## 3. 客户端数据获取 (client)

**影响：** 中高  
**描述：** 自动去重和高效的数据获取模式可以减少冗余网络请求。

## 4. 重渲染优化 (rerender)

**影响：** 中等  
**描述：** 减少不必要的重渲染可以最小化浪费的计算并提高 UI 响应性。

## 5. 渲染性能 (rendering)

**影响：** 中等  
**描述：** 优化渲染过程可以减少浏览器需要完成的工作。

## 6. JavaScript 性能 (js)

**影响：** 中低  
**描述：** 热路径的微优化可以累积成有意义的改进。

## 7. 高级模式 (advanced)

**影响：** 低  
**描述：** 需要仔细实现的特定场景的高级模式。
