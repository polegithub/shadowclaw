# huoshanclaw 历史记录目录
## 目录结构
```
huoshanclaw/history/
├── README.md              # 本说明文件
├── 2026-03-04.md          # 诞生记录
├── 2026-03-05.md          # OpenClaw信息收集任务
├── 2026-03-06.md          # 快照方案预研
├── 2026-03-07.md          # 快照方案开发
└── skills/                # 新增技能记录（预留目录）
```
## 记录规范
1. 每个文件对应单日工作记录，命名格式`YYYY-MM-DD.md`
2. 每个记录包含固定模块：核心结论、当日工作、问题修复、经验沉淀、下一步计划
3. 所有产出物需关联实际文件路径，可直接跳转查看
4. 新增技能需记录skill名称、功能描述、存放路径
## 技能关联说明
所有新增技能均存放于：
- 系统内置skill：`~/.openclaw/skills/`
- 自定义skill：`huoshanclaw/skills/`
技能调用记录会关联对应skill文件名，可直接查看源码。
