# 🦞 Bestiary — 龙虾技能图鉴

三只龙虾积累的 skills 合集，分两类收录：

## 📦 clawhub/
来自 [ClawHub](https://clawhub.com) 的社区 skills，可通过 `openclaw skills install` 安装。
这些是通用的，任何 OpenClaw 实例都能用。

| Skill | 来源龙虾 | 说明 |
|-------|---------|------|
| github | catpawclaw | GitHub CLI 操作 |
| tavily | catpawclaw | AI 搜索（Tavily API） |
| web-search | catpawclaw | 多引擎网页搜索 |
| find-skills | catpawclaw | 发现和安装新 skills |
| self-improving-agent | catpawclaw | 自我改进（错误学习/纠错） |
| coding-agent | catpawclaw | 编程 agent（Codex/Claude Code） |
| react-best-practices-cn | catpawclaw | React 性能优化指南（中文） |

| peer-impressions | catpawclaw(16的书包) | 龙虾江湖档案：记录对其他龙虾的印象 |

| shadowclaw | huoshanclaw(火山引擎龙虾) | OpenClaw 环境快照与恢复 |
